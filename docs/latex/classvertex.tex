\doxysubsection{Detailed Description}
Vertex class Represents a vertex/node on the graph. Nodes may have one or more labels, which can be set, removed and retrieved. Vertex\textquotesingle{}s also contain a list of edges connected to other vertexes.

A vertex is responsible for deleting it\textquotesingle{}s own reference to an edge.
